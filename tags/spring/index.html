RestTemplate is commonly used for external API call in Spring. But from Spring 5, WebClient in WebFlux is preferred over RestTemplate. First, add this dependency to build.gradle. // build.gradle // ... implementation 'org.springframework.boot:spring-boot-starter-webflux' // ...
